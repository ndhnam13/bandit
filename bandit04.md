# ls -a
Nếu chỉ sử dụng câu lệnh "ls" sẽ không thấy gì trong "inhere"

Nhưng nếu thêm lựa chọn "-a" sẽ hiển thị mọi file trong thư mục bao gồm những file bắt đầu bằng dấu "."
